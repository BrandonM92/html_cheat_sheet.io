HTML & CSS Cheat Sheet

Tag/Attribute Description Example
<html> Defines the root of an HTML document There can only be one of these in a HTML File
<head> Contains meta-information about the document There can only be one of these in a HTML File
<title> Defines the title of the document <title>My Website</title>
<body> Defines the body of the document There can only be one of these in a HTML File
<span> Defines a section in a document for styling purposes <span style="color: red;">Highlighted text</span>
id Specifies a unique id for an HTML element <div id="unique-id"></div>
class Specifies one or more class names for an HTML element <div class="my-class"></div>
style Specifies inline CSS for an HTML element <p style="color: red; font-size: 16px;">Styled paragraph</p>
href Specifies the URL of the linked resource <a href="https://www.example.com">Link</a>
src Specifies the URL of an external script or resource <img src="image.jpg" alt="Image">
width Specifies the width of an element <img src="image.jpg" width="100">
height Specifies the height of an element <img src="image.jpg" height="100">
color Specifies the text color <p style="color: blue;">Blue text</p>
background-color Specifies the background color <div style="background-color: yellow;">Yellow background</div>
<video> Embeds video content into a document <video src="video.mp4" controls></video>
<img> Embeds an image into a document <img src="image.jpg" alt="Image">
<hr> Represents a thematic break between paragraphs <hr>
<br> Inserts a single line break Line 1<br>Line 2
Boilerplate Common starting point for an HTML document <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>

HTML Table Creation & Styling

Tag/Attribute Description Example
<table> Defines an HTML table <table> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr> </table>
<tr> Defines a row in an HTML table <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr>
<td> Defines a cell in an HTML table <td>Row 1, Cell 1</td>
border Specifies the border width of the table <table border="1">
cellpadding Specifies the padding within each cell <table cellpadding="5">
cellspacing Specifies the spacing between cells <table cellspacing="10">
bgcolor Specifies the background color of the table <table bgcolor="#ffeebb">
colspan Specifies the number of columns a cell should span <td colspan="2">
rowspan Specifies the number of rows a cell should span <td rowspan="2">
CSS Styling You can apply CSS to further style the table <style>
table {
  border-collapse: collapse;
}
th, td {
  border: 1px solid #dddddd;
  padding: 8px;
  text-align: left;
}
th {
  background-color: #f2f2f2;
}
.highlight {
  background-color: #b1b1b1c2;
}
</style>

CSS Selectors and Specificity

Selector Description Example Specificity
Element Selector Selects elements based on their tag name p { color: blue; } Low
Class Selector Selects elements with a specific class attribute .my-class { font-size: 16px; } Medium
ID Selector Selects a single element with a specific id attribute #unique-id { background-color: yellow; } High
Descendant Selector Selects elements that are descendants of another element ul li { list-style-type: none; } Higher
Adjacent Sibling Selector Selects an element that is directly adjacent to another element h2 + p { font-weight: bold; } Higher
Child Selector Selects elements that are direct children of another element ul > li { padding-left: 20px; } Higher
Pseudo-class Selector Selects elements based on their state or position a:hover { text-decoration: underline; } Medium
Inline Styles Styles applied directly to an element via the style attribute <div style="color: green;"></div> Highest

Link Properties

Name of Attribute Description Example in CSS
color Specifies the color of the link text color: blue;
text-decoration Specifies the decoration added to the link text text-decoration: underline;
:hover Selects links on mouse over .link-example:hover
:active Selects links on mouse click .link-example:active
:visited Selects visited links .link-example:visited
text-decoration Specifies the decoration removed on hover text-decoration: none;
color Specifies the color of the link text on hover color: red;
font-weight Specifies the weight of the link text on hover font-weight: bold;
